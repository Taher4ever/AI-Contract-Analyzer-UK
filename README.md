# ContractLens AI

AI-powered SaaS that explains UK legal documents in plain English — upload a contract, get a risk score, summary, clause analysis, timeline, and chat with AI about it.

## Setup

```bash
git clone https://github.com/Taher4ever/AI-Contract-Analyzer-UK.git
cd AI-Contract-Analyzer-UK
npm install
cp .env.example .env.local   # fill in your keys
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
