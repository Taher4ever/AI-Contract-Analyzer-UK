# Phase 12 — Export & shareable links

**Depends on:** Phase 9.
**Goal:** Professional PDF report, Word report, and public share link for any analysis.

## Tasks

1. Install `@react-pdf/renderer` (PDF) and `docx` (Word).
2. **PDF report** `lib/export/pdf-report.tsx` + `app/api/export/pdf/[contractId]/route.ts`:
   - Branded document: cover (logo, contract title, date, risk score badge), then sections — Summary, Risk breakdown, each findings section (severity-colored bullets), Notice Period / Auto-Renewal facts, Missing Clauses, Timeline table, Recommended Questions. Footer on every page: "Generated by ContractLens AI — not legal advice" + page numbers.
   - Route: auth + ownership → stream file, `Content-Disposition: attachment; filename="{title}-analysis.pdf"`.
3. **Word report** `lib/export/docx-report.ts` + `app/api/export/docx/[contractId]/route.ts`: same content using `docx` (headings, tables, colored severity text).
4. **Share link**:
   - Server action `createShareLink(contractId)` → insert `shared_links` row with random 32-char token; `revokeShareLink`.
   - `app/share/[token]/page.tsx`: public, read-only analysis view — reuse Phase 9 report components (no chat, no actions, no contract file download; contract text pane included read-only). Banner: "Shared analysis · ContractLens AI" + CTA "Analyze your own contract". `noindex` robots meta. 404 if token invalid/revoked/expired.
   - Uses admin client server-side to fetch by token (bypasses RLS deliberately, read-only fields only).
5. **Export UI**: "Export" button in the analysis page header → dropdown: Download PDF, Download Word, Copy share link (creates if none, copies URL, success toast + revoke option in a small dialog).
6. Free plan: exports allowed (keep simple); share links Pro-only → show upgrade dialog for free users (reuse Phase 7 dialog).

## Acceptance criteria

- [ ] PDF and DOCX download, open correctly (test both), contain all sections with branding.
- [ ] Share link opens the analysis logged-out; revoked token 404s.
- [ ] Free user hitting share gets the upgrade dialog.
- [ ] `npm run build` passes.
